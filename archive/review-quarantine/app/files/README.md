## Test Format (2025 Version)

- **Total Question Bank**: 128 questions
- **Questions Asked**: Up to 20
- **Passing Score**: 12 correct (60%)
- **Test Type**: Oral examination
- **Special Consideration (65/20)**: 
  - Only 20 marked questions to study
  - 10 questions asked
  - 6 correct needed to pass (60%)

## Study Tips

1. **Daily Practice**: Spend 10-15 minutes each day
2. **Use Visual Aids**: The emojis and mnemonics help memory
3. **Focus on Weak Areas**: The app tracks which questions you miss
4. **Family Study**: Learn together for motivation
5. **Interview Practice**: Don't just memorize - practice speaking answers
